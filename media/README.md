# Carpeta de medios

Acá va el **video tutorial** que se muestra en la Guía de uso (sección "Video tutorial").

## Cómo subir el video

1. Grabá el tutorial de cómo subir los cuestionarios a tu plataforma (Schoology, Moodle o Google Classroom).
2. Exportalo en formato **`.mp4`** (recomendado: H.264 + AAC, para máxima compatibilidad en navegadores).
3. Subí el archivo a esta carpeta con este nombre exacto:

   ```
   media/tutorial-subir-cuestionarios.mp4
   ```

4. Listo. El video aparece automáticamente en la Guía de uso → pestaña **"Video tutorial"**.

> Si querés usar otro nombre o agregar más videos, editá la sección `video` en `js/guia.js`.

## Recomendaciones

- Tamaño: tratá de mantenerlo por debajo de ~50 MB para que cargue rápido (GitHub limita archivos a 100 MB).
- Resolución: 1080p es más que suficiente.
- Si el video es muy pesado, considerá subirlo a YouTube/Drive y enlazarlo en su lugar.
