Fichas de Scratch
y micro:bit.

Escribí el código como texto plano y se dibuja con colores: Scratch como bloques en la versión que elijas (Scratch 3, Scratch 2 o alto contraste, cada una con su paleta oficial), micro:bit convirtiendo el código JavaScript de MakeCode en bloques automáticamente, y cualquier lenguaje de texto (Python, JavaScript, Java, C++, HTML, SQL...) con resaltado de sintaxis y detección automática del lenguaje. Cada ficha lleva consigna, código, imagen opcional y notas, y el documento final se descarga en PDF o Word listo para repartir en clase.

PASO 01 Datos del documento Poné título y grupo, y elegí cómo se ve el documento: numeración, bordes, una ficha por página.
PASO 02 Creá las fichas A mano, desde una plantilla, pegando texto (varias de una vez) o con ayuda de una IA. Todo se edita después.
PASO 03 Descargá o convertí PDF para imprimir, Word para editar — o convertí las fichas en un cuestionario para CREA con las imágenes de bloques en las preguntas.

Fichas en lote
desde Texto

Para crear muchas fichas de una vez. Escribí (o generá con una IA) un solo texto con todas las fichas y pegalo acá. Cada ficha empieza con === FICHA: Título ===. También sirve para editar rápido: con «Copiar como texto» (en Descargar) bajás las fichas actuales, las retocás en cualquier editor y las volvés a pegar.
Ver el formato completo (referencia)

Líneas que entiende cada ficha (solo codigo: es obligatoria para que se dibuje algo):

  • === FICHA: Título === — empieza una ficha nueva
  • tipo: scratch (por defecto) o microbit
  • version: solo Scratch — scratch3, scratch2 o alto contraste
  • lenguaje: solo micro:bit — javascript o python
  • muestra: solo micro:bit — bloques, codigo o ambos
  • consigna: el enunciado para el alumno (puede seguir en varias líneas)
  • codigo: — y el código en las líneas siguientes
  • notas: pista o aclaración que va al pie de la ficha

Antes de la primera ficha podés poner titulo: y nivel: para el documento.

// Paso 4 (opcional) — Convertir en cuestionario para CREA

Tus fichas también sirven como cuestionario para CREA, Schoology o Moodle: cada pregunta lleva la imagen de los bloques como parte del enunciado. El cuestionario se abre en el Editor de siempre, donde lo revisás, agregás o quitás preguntas y descargás el .imscc con las imágenes adentro.
