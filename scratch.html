Fichas de Scratch
y micro:bit.

Escribí el código como texto plano y se dibuja como bloques de colores. Para Scratch podés elegir la versión (Scratch 3, Scratch 2 o alto contraste, cada una con su paleta oficial); para micro:bit el código JavaScript de MakeCode se convierte en bloques automáticamente. Armá fichas con consigna, código e imagen opcional, y descargalas en PDF o Word.

¿Cómo se escribe el código? Ver ejemplos

Escribí cada bloque en una línea, en español o inglés, tal como se lee en Scratch. Ejemplos:

al presionar bandera verde
por siempre
  mover (10) pasos
  si <¿tocando un borde?> entonces
    girar (180) grados
  fin
fin
  • (10) — números y valores redondeados van entre paréntesis
  • <condición> — los bloques de condición (hexágono) van entre < >
  • [hola] — los textos van entre corchetes; [v] al final marca un desplegable: fijar tamaño a (100) % [v]
  • fin — cierra los bloques «si», «por siempre», «repetir»
  • Podés separar dos pilas de bloques con una línea en blanco

Fichas de micro:bit: escribí el código JavaScript de MakeCode (el mismo que ves en la pestaña JavaScript de makecode.microbit.org) y los bloques se dibujan solos con el servicio oficial de MakeCode — necesita conexión a internet. Podés elegir si la ficha muestra los bloques, el código o ambos. Si escribís en Python, se muestra el código tal cual.

También podés pedirle el código a una IA con el botón «Prompt para IA» de cada ficha.
